<span data-ttu-id="e8472-132">クエリの変換とコンパイルにはコストがかかるため、EF Core ではコンパイル済みのクエリ プランをキャッシュします。</span><span class="sxs-lookup"><span data-stu-id="e8472-132">Since query translation and compilation are
