## <a name="avoiding-dbcontext-threading-issues"></a><span data-ttu-id="8e6f6-144">DbContext スレッドの問題の回避</span><span class="sxs-lookup"><span data-stu-id="8e6f6-144">Avoiding DbContext threading issues</span></span>

<span data-ttu-id="8e6f6-148">EF Core が `DbContext` インスタンスを同時に使用しようとしたことを検出すると、次のようなメッセージが `InvalidOperationException` 表示されます。</span><span class="sxs-lookup"><span data-stu-id="8e6f6-148">When EF Core detects an attempt to use a `DbContext` instance concurrently, you'll see an `InvalidOperationException` with a message like this:</span></span>

<span data-ttu-id="8e6f6-151">同時アクセスが検出されない場合は、未定義の動作、アプリケーションのクラッシュ、データの破損が発生する可能性があります。</span><span class="sxs-lookup"><span data-stu-id="8e6f6-151">When concurrent access goes undetected, it can result in undefined behavior, application crashes and data corruption.</span></span>
